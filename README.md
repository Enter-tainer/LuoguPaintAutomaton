# 使用
请在`cookies.txt`中用以下格式存放cookie
``` plain
UM_id client_id
UM_id client_id
...

```
在源码中可以指定图片的左上角坐标，修改`base_x base_y`即可
## 关于图片
请首先将图片转换为`ppm`格式，**建议使用GIMP**

然后修改源码中的文件名就好了

请使用`python3`，并确保安装`requests`库
